# Cómo contribuir al Código Semilla

Gracias por tu interés en revisar, corregir o fortalecer el **Código Semilla v0.1-RC4** del **Instituto de Expansión Humana Intergaláctica**.

Esta versión se encuentra en revisión pública global. No es una versión final.

## Formas de contribuir

Puedes contribuir mediante:

* observaciones en Issues;
* correcciones científicas;
* mejoras de lenguaje;
* propuestas éticas;
* recomendaciones legales;
* mejoras de traducción;
* propuestas de nuevos Núcleos Semilla;
* detección de áreas faltantes;
* sugerencias de estructura o gobernanza.

## Reglas básicas

Toda contribución debe ser:

1. respetuosa;
2. clara;
3. argumentada;
4. trazable;
5. orientada a fortalecer el documento;
6. abierta a revisión.

No se aceptarán ataques personales, propaganda política, apropiación privada del proyecto, lenguaje discriminatorio o propuestas que contradigan la misión abierta del Código Semilla.

## Versión canónica

El documento principal en español es:

`CODIGO_SEMILLA_v0.1-RC4.es.md`

Esa versión es el punto de referencia inicial.

El idioma canónico inicial es el español porque el documento fundador fue concebido, redactado y estructurado originalmente en español. Esto no excluye otros idiomas. El proyecto podrá recibir aportes en cualquier lengua.

## Cómo enviar una revisión

Para enviar una observación, puedes usar la plantilla:

`revision-v0.1/PLANTILLA_DE_REVISION.md`

También puedes abrir un Issue nuevo explicando:

* sección relacionada;
* problema detectado;
* justificación;
* fuente o referencia, si aplica;
* propuesta de cambio;
* idioma original del aporte.

## Protección del Código Semilla

Todos pueden leer.
Todos pueden estudiar.
Todos pueden proponer.
Nadie puede modificar directamente el núcleo oficial sin revisión.

El objetivo no es bloquear aportes.
El objetivo es proteger la coherencia, trazabilidad y continuidad del Código Semilla.
