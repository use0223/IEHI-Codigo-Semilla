# Mensaje público inicial

**Fecha de emisión:** 29 de junio de 2026
**Versión:** Código Semilla v0.1-RC4
**Institución:** Instituto de Expansión Humana Intergaláctica
**Programa maestro:** Protocolo Semilla
**Estado:** candidato inicial para revisión pública global
**Periodo sugerido de revisión:** 90 días
**Objetivo de la revisión:** preparar una versión más sólida: Código Semilla v1.0

---

Se abre públicamente el **Código Semilla v0.1-RC4** del **Instituto de Expansión Humana Intergaláctica**.

Esta versión no se presenta como una verdad final, una institución terminada ni una promesa de misiones inmediatas.

Se presenta como una base abierta para revisión científica, ética, técnica, legal, lingüística y organizativa.

El objetivo del Código Semilla es organizar, preservar y acelerar el conocimiento necesario para la continuidad y expansión responsable de la humanidad más allá de la Tierra.

En esta etapa se busca revisión crítica, no aprobación ciega.

Se solicitan observaciones sobre:

* errores científicos;
* lenguaje exagerado o ambiguo;
* riesgos éticos;
* áreas faltantes;
* problemas de gobernanza;
* propuestas técnicas;
* traducciones;
* protección del código abierto;
* estructura institucional;
* mecanismos para evitar privatización del conocimiento;
* formas de mejorar la transición hacia el Código Semilla v1.0.

## Principio de esta etapa

No se pide fe.
No se pide obediencia.
No se pide aprobación vacía.

Se pide revisión, crítica útil, corrección y construcción colectiva.

El conocimiento generado bajo esta misión deberá permanecer abierto, verificable, documentado y disponible para la humanidad.

Los resultados pertenecen a la humanidad.
El reconocimiento pertenece a quienes los hacen posibles.

---

La Tierra será el origen.
La galaxia será el camino.
La semilla humana no debe morir en un solo mundo.
