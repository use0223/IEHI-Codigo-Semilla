# Divulgación

Esta carpeta reunirá materiales públicos de divulgación del Código Semilla y del Instituto de Expansión Humana Intergaláctica.

Su función será facilitar que el proyecto llegue a personas capaces de revisar, corregir, traducir, explicar o fortalecer el documento.

Los materiales de divulgación no reemplazan el Código Semilla canónico.  
Su función es abrir puertas hacia el documento principal.

## Contenido previsto

- mensajes públicos;
- textos cortos para redes;
- comunicados;
- imágenes;
- material gráfico;
- versiones resumidas;
- preguntas frecuentes;
- enlaces de presentación.

## Regla de divulgación

La divulgación deberá ser clara, responsable y verificable.

No deberá presentar el Código Semilla como una versión final, una institución terminada o una promesa de misiones inmediatas.
