# Traducciones

Esta carpeta reunirá las versiones traducidas del Código Semilla.

## Idioma canónico inicial

El idioma canónico inicial del Código Semilla será el español porque el documento fundador fue concebido, redactado y estructurado originalmente en español.

Esta decisión no representa superioridad lingüística, exclusión cultural ni limitación internacional. Su función es mantener una referencia inicial única para controlar versiones, evitar contradicciones, revisar cambios y sincronizar traducciones.

El Instituto de Expansión Humana Intergaláctica reconoce que una misión perteneciente a la humanidad no puede quedar limitada por un solo idioma.

Por ello, el proyecto podrá recibir aportes en cualquier idioma y buscará desarrollar versiones sincronizadas del Código Semilla en múltiples lenguas.

## Reglas de traducción

Toda traducción relevante deberá conservar:

1. idioma original;
2. traducción asistida cuando sea necesario;
3. revisión humana o técnica;
4. fecha de actualización;
5. relación con el documento canónico;
6. estado de revisión.

Ninguna traducción automática será considerada canónica sin revisión humana o técnica cuando afecte decisiones científicas, éticas, legales u operativas.

## Principio lingüístico

El español será el punto de partida documental.
La humanidad será el destino comunicativo.
