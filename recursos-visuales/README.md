# Recursos visuales

Esta carpeta reunirá recursos gráficos del Instituto de Expansión Humana Intergaláctica y del Código Semilla.

## Contenido previsto

- logotipos;
- emblemas;
- versiones transparentes;
- banners;
- imágenes de divulgación;
- material para documentos;
- material para redes sociales.

## Regla de uso

Los recursos visuales deberán usarse para representar el proyecto con claridad, seriedad y responsabilidad.

Ningún material visual deberá presentar el Código Semilla como una versión final, una institución terminada o una promesa de misiones inmediatas.
